PROJECTS

Healthy Living

This project was built for a client using React. The project takes advantage of React's Component based architecture and renders both statefull Class based components and stateless Functional components. One cool element is the connection to the Google API that renders live news specific to this clients business. The design layout uses both CSS FlexBox and Grid.

Breast Connections

This project built in React was part of my group project at DevMountain. The idea was inspired by one of our team mates mother that is a breast cancer survivor. The goal was to provide a user based application for cancer survivors and those still fighting cancer to find support and other resources more easily. The project uses B-Crypt for authentication. Users can create posts and post them in a variety of forums. Users can like other users posts, as well as direct message other users. Users can perform live searches for Doctors and other resources through the Google API which returns images and maps for each result. It contains an express server and a database built with sqltabs and hosted on Heroku. The design layout includes CSS FlexBox and Grid as well as the library from Ant-d.
