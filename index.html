Breast Connections

This project built in React is a full CRUD application and was part of my group project at DevMountain. The idea was inspired by one of our teammates mother that is a breast cancer survivor. The goal was to provide a user based application for cancer survivors and those still fighting cancer to find support and other resources more easily.

The project uses B-Crypt for authentication. Users can create posts and post them in a variety of forums. Users can like other users posts, as well as direct message other users. Users can perform live searches for Doctors and other resources through the Google API which returns images and maps for each result. Users can also write reviews on the returning data. Breast Connections contains an express server and a database built with MYsql and hosted on Heroku. The design layout uses reusable design components that pass props as well as CSS FlexBox and Grid. We also took advantage of the awesome React based design library from Ant-d. Here is the link to the github repository.
